Overview

  

•  The I-CUBE-BAIDU (STM32 BAIDU IoT software expansion for STM32Cube) provides ready-to-run firmware examples to support quick evaluation and development of BAIDU IoT Cloud applications on B-L475E-IOT01, and 32L496GDISCOVERY board connected to an external Quectel UG96 cellular module.

•  3 IDE are supported: IAR, Keil and System Workbench.

•  It contains BSP compatible Sensor drivers for B-L475E-IOT01.

•  It connects to BAIDU IoT hubs through an MQTT on TLS connection.
